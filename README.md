# Assessment Based Job Discovery Agent

## Objective

An AI-powered system that discovers companies where hiring decisions are based on skill assessments rather than only resume screening.

## Features Planned

- Company data collection
- Job opening discovery
- Assessment platform detection
- Fresher Data Analyst role filtering
- Hugging Face AI classification
- Excel report generation

## Technologies

- Python
- BeautifulSoup
- Requests
- Pandas
- Hugging Face Transformers
- OpenPyXL
- GitHub
